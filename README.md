 

### Selfy Client in React Native
========

working with:
```
npm 3.8.9
node v6.2.0
react-native-cli: 1.0.0
react-native: 0.34.0
 ```

Download and install node modules.
```
npm install
```

Run client version.
```
react-native run-ios
```
tested on ios10.0 iphone6
```
react-native run-android
```
